## Project description

A book recommendation system that will give the user books that he might like.

## Installation

pip install -r requirements.txt

- Python version: 3.13.3
- Pandas version: 2.3.3
- NumPy version: 2.3.4
- Scikit-learn version: 1.7.2
- Streamlit version: 1.51.0
- SpeechRecognition version: 3.14.3
- joblib: 1.5.2
- nltk: 3.9.2
- pyttsx3: 25.3
- gTTS: 2.5.4

## Basic Usage

The user can add his desired book ratings, if it's popular or not and the length that he desires.

The Use Case that can be searched for in the system by the user relates to Top-Rated Books.
